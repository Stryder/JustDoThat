########### INSTALL ###########

1. Récupérer les sources (c'est fait logiquement ^^) 
2. Lancer WAMP (ou MAMP/LAMP) ou MySQL si vous avez PHP en natif sur votre machine
3. creer une BDD nommé IMAC3_ProjetDjango
4. Editer settings.py pour ajouter votre config BDD
5. faire un runserver pour verifier qu'il n'y a pas de soucis
6. Puis un validate
7. Puis un syncdb
7bis. Ajouter un utilisateur quand cela vous sera demandé dans la console
8. Accéder à localhost:8000/admin/ (ou 127.0.0.1:8000/)

########### ADMIN ###########
Il suffit d'accéder à localhost:8000/admin/
